Base De Datos - Sanidad

El proyecto consiste en el desarrollo de una aplicación web para la gestión de una base de datos sanitaria, implementada utilizando el stack XAMPP (Apache, MySQL, PHP) y desarrollada en Visual Studio Code. La aplicación permite dar de alta y administrar entidades clave del entorno sanitario, como pacientes, personal médico, especialidades y hospitales. Además, incluye un sistema de autenticación de usuarios con funciones de registro y login seguro, que controla el acceso al sistema según el tipo de usuario. La interfaz fue desarrollada con HTML, CSS y JavaScript, mientras que la lógica de negocio y la conexión con la base de datos MySQL se gestionan mediante PHP. El diseño modular de la aplicación facilita su mantenimiento y escalabilidad para futuras funcionalidades.

Pagina Web - MaoMao

Tienda web desarrollada en WordPress con Astra y Elementor para una marca de ropa inspirada en la estética Visual Kei ypostapocalíptica. Incluye catálogo de productos, filtros por categoría y precio, descripciones con storytelling y diseño responsive. Hecha solo con herramientas gratuitas y recursos libres, enfocada en crear una experiencia visual fuerte sin depender de plugins pagos. Tecnologías: WordPress, Astra, Elementor, WooCommerce, CSS básico.


Feel Tracker - Sistema de Terapia de Sistemas de Familia Interna

Aplicacion web personalizada diseñada para el seguimiento diario del estado emocional utilizando el modelo de Terapia de Sistemas de Familia Interna (IFS). Permite registrar qué parte interna está al frente, identificar creencias o sensaciones intensas (representadas como "monstruos") y ofrecer un espacio para responder desde el Self compasivo. Incluye funcionalidades como historial persistente mediante localStorage, eliminación individual de entradas y diseño minimalista enfocado en la autoreflexion. Fue desarrollada desde una perspectiva terapéutica integradora, orientada a la exploracion emocional segura y al desarrollo de la autocompasion. Tecnologías utilizadas: HTML, CSS, JavaScript.
